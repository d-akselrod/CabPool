# TaxiPool Backend

## Setup

### Pre-requisites

- [Node.js](https://nodejs.org/en/download/)
- .env

  ```.env
  PORT=8081
  HOST=localhost
  HOST_URL=http://localhost:8081

  #firebase database config

  API_KEY=
  AUTH_DOMAIN=
  DATABASE_URL=
  PROJECT_ID=
  STORAGE_BUCKET=
  MESSAGING_SENDER_ID=
  APP_ID=
  ```

### Installation

1. npm install
2. npm run start
